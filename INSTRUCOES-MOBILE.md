# Como hospedar sua Bio Page pelo App do GitHub (Celular)

## 📱 Passo a passo pelo aplicativo móvel

### 1. Baixar e instalar o app
- Baixe o app "GitHub" na Play Store ou App Store
- Faça login com sua conta GitHub

### 2. Criar um novo repositório
1. Abra o app GitHub
2. Toque no ícone "+" no canto superior direito
3. Selecione "New repository"
4. Nome do repositório: `shinoffx-biopage`
5. Marque como "Public"
6. Toque em "Create repository"

### 3. Fazer upload dos arquivos (LIMITAÇÃO DO APP)
⚠️ **PROBLEMA**: O app móvel do GitHub tem limitações para upload de múltiplos arquivos.

**SOLUÇÕES ALTERNATIVAS:**

#### Opção A: Usar o navegador do celular
1. Abra o navegador (Chrome, Firefox, etc.)
2. Acesse [github.com](https://github.com)
3. Faça login
4. Vá no seu repositório criado
5. Toque em "uploading an existing file"
6. Selecione os arquivos do ZIP que te enviei
7. Faça commit

#### Opção B: Usar um computador/notebook
- É a forma mais fácil e rápida
- Siga as instruções do README.md principal

#### Opção C: Criar arquivo por arquivo no app (TRABALHOSO)
1. No app, vá no seu repositório
2. Toque em "Create new file"
3. Copie e cole o conteúdo de cada arquivo manualmente
4. Repita para todos os arquivos

### 4. Ativar GitHub Pages pelo app
1. No seu repositório, toque em "Settings"
2. Role para baixo até "Pages"
3. Em "Source", selecione "Deploy from a branch"
4. Em "Branch", selecione "main"
5. Toque em "Save"

### 5. Acessar sua bio page
Após alguns minutos:
`https://[seu-usuario].github.io/shinoffx-biopage`

## 💡 RECOMENDAÇÃO
**Use o navegador do celular** em vez do app para fazer upload dos arquivos. É muito mais fácil e rápido!

## 📋 Arquivos que você precisa fazer upload:
- `index.html`
- `css/styles.css`
- `media/profile.jpg`
- `media/background.png`
- `media/badge2.png`
- `media/badge3.png`
- `media/badge4.png`
